# Skyline Weather App

A modern **Weather Forecast Web Application** built using **HTML, CSS, and JavaScript**.

## ✨ Features

- 🌍 Search weather by city
- 📍 Detect current location
- 🌡️ Current temperature and "feels like"
- 🌤️ 24-hour forecast
- 📅 7-day weather forecast
- 🌫️ Air Quality Index (AQI)
- 🌅 Sunrise and Sunset times
- 🌧️ Dynamic weather animations (Rain, Snow, Storm, Fog, Day & Night)
- 🔄 Celsius / Fahrenheit toggle
- 💾 Saves user preferences using Local Storage
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Open-Meteo Weather API
- Open-Meteo Air Quality API
- Open-Meteo Geocoding API
